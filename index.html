HTML

簡介

一種建立網頁的標準標記語言，用來組織架構並呈現網頁內容的程式語言，常與CSS與JavaScript一起使用在網頁設計上。

基礎程式入門

<!DOCTYPE html>

<html>

<!-- created 2010-01-01 -->

<head>

<title>sample</title>

</head>

<body>

<p>Voluptatem accusantium
totam rem aperiam.</p>

</body>

</html>

 用來代表文件
 告訴瀏覽器說，這是以HTML標記語言所撰
 寫的文件，請用HTML的定義解讀的意思。

CSS

簡介

CSS負責靜態地修飾網頁，或者配合各種語言動態地對網頁各元素進行格式化。 CSS的功用主要是針對於網站的排版部分，支援大多數的字型字號樣式，對網頁中的元素進行位置調整擁有對網頁物件和模型樣式編輯的能力。

基本功能介紹

調整字型設定

font-文字設定

font-family-字體設定

font-size文字大小設定

color-文字顏色設定

font-weight-文字粗細設定

line-height-文字行列間距

設定內外距

border-width-設定邊框寬度

border-style-設定邊框樣式

border-color-調整邊框顏色

padding/margin

(top/right/bottom/right)

-內/外距(上/下/左/右)距離調整

Javascript

簡介

JavaScript可以讓你在網頁中實現出複雜的功能，讓網頁不只呈現靜態的內容， 例如：內容即時更新、地圖交動、繪製 2D/3D 圖形，影片播放控制、與閱覽者互動。
