HTML

簡介

一種建立網頁的標準標記語言，用來組織架構並呈現網頁內容的程式語言，常與CSS與JavaScript一起使用在網頁設計上。

基礎程式入門

<!DOCTYPE html>

<html>

<!-- created 2010-01-01 -->

<head>

<title>sample</title>

</head>

<body>

<p>Voluptatem accusantium
totam rem aperiam.</p>

</body>

</html>

 用來代表文件
 告訴瀏覽器說，這是以HTML標記語言所撰
 寫的文件，請用HTML的定義解讀的意思。

       < html > = 開始的
       < /html > = 結束
       兩者中間為網頁其內容。
